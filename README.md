# UpOnline Landing Page

Landing page for **UpOnline DC** — AI Operations Brain for GPU Data Centers.

> Predict infrastructure risk, protect training jobs, and validate every change — before incidents happen. Purpose-built for AI-scale compute.

## Quick start

Open `UpOnline Landing Page/index.html` in a browser (double-click or run a local server from that folder):

```bash
cd "UpOnline Landing Page"
open index.html

# Or with a local server:
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## What’s included

- **Single-page landing** with hero, problem, demo placeholder, capabilities, how it works, trust section, and early-access form
- **Styles** — All CSS is inline; only external dependency is [Google Fonts](https://fonts.googleapis.com) (DM Sans, JetBrains Mono)
- **Form** — “Request Early Access” submits to [Formspree](https://formspree.io); update the form action in `index.html` if you use a different endpoint
